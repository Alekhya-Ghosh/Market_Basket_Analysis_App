An interactive Streamlit application that uses the Apriori algorithm to identify products frequently purchased together, enabling retailers to improve cross-selling, product bundling, and recommendation strategies.

Tech Stack: Python • Pandas • Streamlit • mlxtend • Plotly

Business Problem
Retailers process thousands of customer transactions every day, but identifying purchasing patterns manually is difficult. This project analyzes transactional data to uncover product associations that support better merchandising, marketing, and inventory decisions.
